# ระบบจัดการเมนูอาหารและสูตร (PHP OOP)

โปรเจกต์แสดงรายการอาหาร หมวดหมู่ และวัตถุดิบสูตรอาหาร เขียนด้วยภาษา PHP ในรูปแบบ OOP (Object-Oriented Programming)

## โครงสร้างไฟล์
* **`week5food.php`** : เก็บข้อมูลเมนูอาหารและหมวดหมู่
* **`week5aecpi.php`** : เก็บสูตรอาหารและวัตถุดิบ
* **`menu.php`** : ดึงข้อมูลจาก 2 ไฟล์แรกมาประมวลผลและแสดงผล
